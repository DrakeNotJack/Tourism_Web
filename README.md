# Manyou Holiday - Tourism Website
A responsive tourism website built with HTML5, CSS3, Bootstrap 5, and JavaScript. It includes core modules such as a homepage, service introduction, travel packages, about us, and contact form.

## Project Structure
Tourism_Web/
├── index.html # Homepage
├── services.html # Services page
├── portfolio.html # Travel packages page
├── about.html # About us page
├── contact.html # Contact page
├── CSS/
│ └── style.css # Custom styles
├── js/
│ ├── script.js # Interactive logic
│ └── main.js # Form submission handler
└── images/ # Image assets
plaintext

## How to Access the Website

### Option 1: Local Access
1. Clone or download the repository to your local machine
2. Locate the HTML files (e.g., `index.html`, `services.html`) in the project root
3. Double-click the HTML file to open it in your default browser
   - Alternatively, right-click the file → "Open with" → select any browser (Chrome, Firefox, Safari, etc.)

### Option 2: Online Access (GitHub Pages)
This website is deployed via GitHub Pages for public access:  
[Manyou Holiday Tourism Website](https://drakenotjack.github.io/Tourism_Web/)

## Key Features
- Responsive navigation bar (mobile/desktop compatible)
- Homepage image carousel
- Categorized travel destination filtering
- Service showcase section
- Interactive contact form (with submission feedback)
- Social media integration

## Tech Stack
- HTML5 (Semantic markup)
- CSS3 (Flexbox/Grid + Bootstrap 5)
- JavaScript (Vanilla JS for interactivity)
